# Mechanical Design

## Dimensions
Approx 20cm x 30cm

## Mass
~1.66kg total

## Shape
The robot uses angled legs with triangular holes to reduce weight and material cost
We want the robot to be relatively tall as this will make it easier to balance, yet not so tall that it could be balanced by a human as this is the task we want to automate.

## Issues
- Wheel misalignment → vibration
- Weak axle
- Loose electronics mounting
